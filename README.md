# BUY SUN Landing Page

Landing page для BUY SUN с hero секцией, адаптированной под все размеры экранов.

## Установка

```bash
npm install
```

## Запуск разработки

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Структура

- `src/Hero.jsx` - компонент hero секции
- `src/Hero.css` - стили hero секции с пропорциональным масштабированием
- `public/man1.png` - изображение для hero секции
